# 🎮 2-Level Mini-Game

A 2-level Unity mini-game built as a final assignment.

## 🕹️ Levels

### Level 1: Light Arena Challenge
Obey the traffic light system while running to the finish line:
- 🟢 Green → Run
- 🔴 Red → Stop immediately
- 🟡 Yellow → Dance

NPC bots follow the same rules but occasionally make mistakes and die.
Reach the finish line to advance to Level 2.

### Level 2: Moving Blocks Challenge
Cross a path full of moving platforms without falling off.
Reach the end → character dances → YOU WIN!

## ✅ Features
- CharacterController with Idle, Run, Dance, Death animations
- Cinemachine Virtual Camera following the player
- TextMeshPro UI (Game Over, Win, Level Complete)
- Scene switching with Unity SceneManager
- NPC bots with random mistake logic
- 10+ moving obstacles with various movement types

## 🛠️ Built With
- Unity (3D)
- C#
- Cinemachine
- TextMeshPro
